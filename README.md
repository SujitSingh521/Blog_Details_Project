# 🌟 Blog Details Project

## 🚀 Quick Setup

### Backend 🛠️
1. Navigate to the backend directory:
   ```bash
   cd Blog_Details_Project/backend

2. Initialize package.json:-
   npm init -y

3. Install dependencies:-
   npm install express mongoose cors dotenv nodemon



(1). express: Framework for building the server.
(2). mongoose: MongoDB ODM library.
(3). cors: Enable CORS (Cross-Origin Resource Sharing).
(4). dotenv: Load environment variables.
(5). nodemon: Auto-restart server during development.




Frontend 🌐 :- 

1. Navigate to the root project directory: -
   cd Blog_Details_Project
   
3. Create a React app:-
   npx create-react-app frontend


4. Navigate to the frontend directory:-
   cd frontend

5. Install dependencies:-
    npm install axios react-router-dom

  (i)  axios: For HTTP requests.
  (ii) react-router-dom: For routing


🎯 Get Started
1. Backend: Run the server:-
   nodemon server.js

2. Frontend: Start the development server:-
   npm start

BLOG PROJECTS 🌐✍️
Objective
Create a full-stack "Blog Details" app using React.js for the frontend and Node.js, Express, MongoDB for the backend to enable CRUD operations on blog posts.




Requirements
Frontend 🎨:

Create, View, Update, and Delete blog posts.
Responsive UI styled with CSS/Bootstrap.
Use Axios for API calls.
Backend ⚙️:

API Endpoints for CRUD operations:
GET /posts 📝
POST /posts ➕
PUT /posts/:id ✏️
DELETE /posts/:id ❌
Use Mongoose for database interaction.
Database (MongoDB) 🗂️:

BlogPost schema fields:
Title, Author, Content, Tags, Published Date.
Routing & Communication 🔗:

Use React Router for navigation.
Axios or Fetch API for communication.
